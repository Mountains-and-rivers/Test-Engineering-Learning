# 13_测试职业进阶能力模型

## 模块简介

本模块系统性地梳理测试工程师从初级到中高级、再到测试负责人的完整职业发展路径，涵盖各阶段的能力模型、技能要求、面试准备和简历优化策略，为测试工程师的职业规划和跳槽晋升提供实战指导。

---

## 目录结构

```
13_测试职业进阶能力模型/
├── README.md                # 本文件
├── 01_初级工程师.md          # 0-2年：执行能力、工具使用、Bug管理基础
├── 02_中级工程师.md          # 2-5年：独立负责、自动化测试、风险识别
├── 03_高级工程师.md          # 5-8年：测试架构设计、技术选型、质量Owner
├── 04_测试负责人.md          # 8年+：团队管理、质量体系搭建、跨部门协作
└── 05_简历优化.md            # STAR法则、ATS优化、项目经验撰写、面试引导
```

---

## 学习路线图

```
第1步: 01_初级工程师       明确入门阶段的核心能力和成长目标
  │
第2步: 02_中级工程师       掌握独立负责和自动化测试的核心技能
  │
第3步: 03_高级工程师       进阶测试架构设计和质量体系建设能力
  │
第4步: 04_测试负责人       理解从技术走向技术+管理的转型路径
  │
第5步: 05_简历优化         将能力模型转化为有竞争力的简历和面试表现
```

---

## 学习建议

1. 按目录顺序学习，每个阶段的能力都是下一阶段的基础
2. 对照自己当前所处的职业阶段，重点精读对应的能力模型文档
3. 向前看一个阶段的能力要求，提前规划和储备相关技能
4. 学完所有能力模型后，使用 05_简历优化.md 将能力转化为简历和面试亮点
5. 建议学习周期：1-2周
   - 第1周：01-04（各阶段能力模型）
   - 第2周：05（简历优化+实战修改）
