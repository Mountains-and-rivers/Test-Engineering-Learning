# 工具封装层

## 1. HTTP客户端封装

### 通俗释义

HTTP客户端封装就像是给测试框架配了一个"万能遥控器"——无论被测接口是RESTful API、GraphQL还是gRPC，测试用例只需要调用统一的封装方法，不用管底层是用OkHttp还是HttpClient。

### 详细解释

#### HTTP客户端技术选型对比

| 客户端 | 语言 | 特点 | 适用场景 |
|--------|------|------|----------|
| **OkHttp** | Java | 连接池复用、HTTP/2支持、拦截器链 | Android/Java后端项目 |
| **Apache HttpClient** | Java | 功能全面、Cookie管理强、代理支持好 | 企业级Java项目 |
| **RestTemplate** | Java(Spring) | Spring生态集成、同步阻塞 | Spring Boot项目 |
| **WebClient** | Java(Spring5+) | 响应式非阻塞、函数式风格 | 高并发场景 |
| **Requests** | Python | 简洁优雅、Session管理 | Python项目首选 |
| **httpx** | Python | 异步支持、HTTP/2、连接池 | Python异步项目 |

#### OkHttp完整封装（Java）

```java
// HttpClientManager.java — OkHttp封装
import okhttp3.*;
import okhttp3.logging.HttpLoggingInterceptor;
import javax.net.ssl.*;
import java.io.File;
import java.io.IOException;
import java.security.SecureRandom;
import java.security.cert.X509Certificate;
import java.util.Map;
import java.util.concurrent.ConcurrentHashMap;
import java.util.concurrent.TimeUnit;

public class HttpClientManager {
    
    private static final MediaType JSON = MediaType.parse("application/json; charset=utf-8");
    private OkHttpClient client;
    private final Map<String, String> defaultHeaders = new ConcurrentHashMap<>();
    
    // 构建器模式创建HttpClient
    public static class Builder {
        private long connectTimeout = 30;
        private long readTimeout = 30;
        private long writeTimeout = 30;
        private boolean followRedirects = true;
        private boolean followSslRedirects = true;
        private boolean retryOnConnectionFailure = true;
        private int maxIdleConnections = 5;
        private int keepAliveDuration = 5;
        private boolean enableLogging = true;
        private HttpLoggingInterceptor.Level logLevel = HttpLoggingInterceptor.Level.BODY;
        // 忽略SSL证书（仅测试环境）
        private boolean ignoreSSL = false;
        // 代理设置
        private String proxyHost;
        private int proxyPort;
        // Cookie管理
        private CookieJar cookieJar;
        
        public Builder connectTimeout(long seconds) { 
            this.connectTimeout = seconds; return this; 
        }
        public Builder readTimeout(long seconds) { 
            this.readTimeout = seconds; return this; 
        }
        public Builder writeTimeout(long seconds) { 
            this.writeTimeout = seconds; return this; 
        }
        public Builder enableLogging(boolean enable) { 
            this.enableLogging = enable; return this; 
        }
        public Builder ignoreSSL(boolean ignore) { 
            this.ignoreSSL = ignore; return this; 
        }
        public Builder cookieJar(CookieJar jar) { 
            this.cookieJar = jar; return this; 
        }
        
        public HttpClientManager build() {
            return new HttpClientManager(this);
        }
    }
    
    private HttpClientManager(Builder builder) {
        OkHttpClient.Builder clientBuilder = new OkHttpClient.Builder()
            .connectTimeout(builder.connectTimeout, TimeUnit.SECONDS)
            .readTimeout(builder.readTimeout, TimeUnit.SECONDS)
            .writeTimeout(builder.writeTimeout, TimeUnit.SECONDS)
            .followRedirects(builder.followRedirects)
            .followSslRedirects(builder.followSslRedirects)
            .retryOnConnectionFailure(builder.retryOnConnectionFailure)
            .connectionPool(new ConnectionPool(
                builder.maxIdleConnections, 
                builder.keepAliveDuration, 
                TimeUnit.MINUTES
            ));
        
        // 日志拦截器
        if (builder.enableLogging) {
            HttpLoggingInterceptor logging = new HttpLoggingInterceptor(
                message -> LoggerFactory.getLogger("HTTP").info(message)
            );
            logging.setLevel(builder.logLevel);
            clientBuilder.addInterceptor(logging);
        }
        
        // 请求/响应拦截器（统一处理Header、Token等）
        clientBuilder.addInterceptor(new RequestInterceptor());
        clientBuilder.addInterceptor(new RetryInterceptor(3)); // 自动重试
        
        // SSL处理
        if (builder.ignoreSSL) {
            clientBuilder.sslSocketFactory(createInsecureSSLSocketFactory(), 
                new X509TrustManager() {
                    @Override public void checkClientTrusted(X509Certificate[] c, String a) {}
                    @Override public void checkServerTrusted(X509Certificate[] c, String a) {}
                    @Override public X509Certificate[] getAcceptedIssuers() { return new X509Certificate[0]; }
                });
            clientBuilder.hostnameVerifier((hostname, session) -> true);
        }
        
        // Cookie管理
        if (builder.cookieJar != null) {
            clientBuilder.cookieJar(builder.cookieJar);
        }
        
        this.client = clientBuilder.build();
    }
    
    /**
     * 通用请求方法封装
     */
    private Response execute(Request request) {
        try {
            Call call = client.newCall(request);
            return call.execute();
        } catch (IOException e) {
            throw new HttpRequestException("HTTP请求失败: " + request.url(), e);
        }
    }
    
    // ===== GET请求 =====
    public HttpResponse get(String url) {
        return get(url, null, null);
    }
    
    public HttpResponse get(String url, Map<String, String> headers) {
        return get(url, headers, null);
    }
    
    public HttpResponse get(String url, Map<String, String> headers, 
                            Map<String, String> params) {
        // 构建URL参数
        if (params != null && !params.isEmpty()) {
            HttpUrl.Builder urlBuilder = HttpUrl.parse(url).newBuilder();
            params.forEach(urlBuilder::addQueryParameter);
            url = urlBuilder.build().toString();
        }
        
        Request.Builder builder = new Request.Builder().url(url).get();
        applyHeaders(builder, headers);
        
        Response response = execute(builder.build());
        return new HttpResponse(response);
    }
    
    // ===== POST请求 =====
    public HttpResponse post(String url, String jsonBody) {
        return post(url, null, jsonBody);
    }
    
    public HttpResponse post(String url, Map<String, String> headers, String jsonBody) {
        RequestBody body = RequestBody.create(jsonBody, JSON);
        Request.Builder builder = new Request.Builder().url(url).post(body);
        applyHeaders(builder, headers);
        return new HttpResponse(execute(builder.build()));
    }
    
    // ===== POST表单 =====
    public HttpResponse postForm(String url, Map<String, String> formData) {
        FormBody.Builder formBuilder = new FormBody.Builder();
        formData.forEach(formBuilder::add);
        Request.Builder builder = new Request.Builder().url(url).post(formBuilder.build());
        return new HttpResponse(execute(builder.build()));
    }
    
    // ===== PUT请求 =====
    public HttpResponse put(String url, String jsonBody) {
        RequestBody body = RequestBody.create(jsonBody, JSON);
        Request.Builder builder = new Request.Builder().url(url).put(body);
        return new HttpResponse(execute(builder.build()));
    }
    
    // ===== DELETE请求 =====
    public HttpResponse delete(String url) {
        Request.Builder builder = new Request.Builder().url(url).delete();
        return new HttpResponse(execute(builder.build()));
    }
    
    // ===== 文件上传 =====
    public HttpResponse upload(String url, File file, String fileKey, 
                              Map<String, String> formFields) {
        MultipartBody.Builder multipartBuilder = new MultipartBody.Builder()
            .setType(MultipartBody.FORM);
        
        if (formFields != null) {
            formFields.forEach(multipartBuilder::addFormDataPart);
        }
        
        MediaType fileMediaType = MediaType.parse(getMimeType(file.getName()));
        multipartBuilder.addFormDataPart(
            fileKey, file.getName(), 
            RequestBody.create(file, fileMediaType)
        );
        
        Request.Builder builder = new Request.Builder()
            .url(url)
            .post(multipartBuilder.build());
        
        return new HttpResponse(execute(builder.build()));
    }
    
    // ===== 辅助方法 =====
    private void applyHeaders(Request.Builder builder, Map<String, String> headers) {
        // 添加默认Header
        defaultHeaders.forEach(builder::addHeader);
        // 添加自定义Header
        if (headers != null) {
            headers.forEach(builder::addHeader);
        }
    }
    
    public void addDefaultHeader(String key, String value) {
        defaultHeaders.put(key, value);
    }
    
    public void setAuthToken(String token) {
        defaultHeaders.put("Authorization", "Bearer " + token);
    }
    
    private String getMimeType(String fileName) {
        if (fileName.endsWith(".json")) return "application/json";
        if (fileName.endsWith(".xml")) return "application/xml";
        if (fileName.endsWith(".pdf")) return "application/pdf";
        if (fileName.endsWith(".png")) return "image/png";
        if (fileName.endsWith(".jpg") || fileName.endsWith(".jpeg")) return "image/jpeg";
        return "application/octet-stream";
    }
}
```

#### 响应封装与断言支持

```java
// HttpResponse.java — 统一响应封装
public class HttpResponse {
    private final int statusCode;
    private final String body;
    private final Map<String, String> headers;
    private final long responseTimeMs;
    private final Response rawResponse;
    
    public HttpResponse(Response response) {
        this.rawResponse = response;
        this.statusCode = response.code();
        this.responseTimeMs = response.receivedResponseAtMillis() - response.sentRequestAtMillis();
        
        // 提取Headers
        this.headers = new HashMap<>();
        response.headers().forEach(pair -> headers.put(pair.getFirst(), pair.getSecond()));
        
        // 读取Body
        try {
            this.body = response.body() != null ? response.body().string() : "";
        } catch (IOException e) {
            throw new HttpRequestException("读取响应体失败", e);
        }
    }
    
    // Getters
    public int getStatusCode() { return statusCode; }
    public String getBody() { return body; }
    public long getResponseTimeMs() { return responseTimeMs; }
    
    // 链式断言方法
    public HttpResponse assertStatus(int expectedStatus) {
        if (statusCode != expectedStatus) {
            throw new AssertionError(String.format(
                "期望HTTP状态码 %d，实际 %d\nResponse Body: %s", 
                expectedStatus, statusCode, body
            ));
        }
        return this;
    }
    
    public HttpResponse assertStatusIn(int... validStatuses) {
        for (int valid : validStatuses) {
            if (statusCode == valid) return this;
        }
        throw new AssertionError(String.format(
            "HTTP状态码 %d 不在期望范围内 %s", 
            statusCode, Arrays.toString(validStatuses)
        ));
    }
    
    public HttpResponse assertBodyContains(String expected) {
        if (!body.contains(expected)) {
            throw new AssertionError(String.format(
                "响应体不包含预期内容: %s\n完整Body: %s", expected, body
            ));
        }
        return this;
    }
    
    public HttpResponse assertResponseTimeLessThan(long maxMillis) {
        if (responseTimeMs > maxMillis) {
            throw new AssertionError(String.format(
                "响应时间 %dms 超过阈值 %dms", responseTimeMs, maxMillis
            ));
        }
        return this;
    }
    
    // JSON解析
    public JsonNode toJson() {
        try {
            return new ObjectMapper().readTree(body);
        } catch (JsonProcessingException e) {
            throw new HttpRequestException("JSON解析失败: " + body, e);
        }
    }
    
    public <T> T toObject(Class<T> clazz) {
        try {
            return new ObjectMapper().readValue(body, clazz);
        } catch (JsonProcessingException e) {
            throw new HttpRequestException("JSON反序列化失败", e);
        }
    }
}
```

#### Python Requests封装

```python
# http_client.py
import time
import json
import logging
from typing import Optional, Dict, Any, Union
from dataclasses import dataclass
import requests
from requests.adapters import HTTPAdapter
from urllib3.util.retry import Retry

logger = logging.getLogger(__name__)

@dataclass
class HttpResponse:
    """统一响应封装"""
    status_code: int
    body: str
    headers: Dict[str, str]
    response_time_ms: float
    raw_response: requests.Response
    
    def json(self) -> Dict[str, Any]:
        return self.raw_response.json()
    
    def assert_status(self, expected: int) -> 'HttpResponse':
        assert self.status_code == expected, \
            f"期望状态码 {expected}，实际 {self.status_code}\nBody: {self.body}"
        return self
    
    def assert_body_contains(self, text: str) -> 'HttpResponse':
        assert text in self.body, \
            f"响应体不包含: {text}\n完整Body: {self.body}"
        return self
    
    def assert_response_time(self, max_ms: float) -> 'HttpResponse':
        assert self.response_time_ms <= max_ms, \
            f"响应时间 {self.response_time_ms}ms 超过阈值 {max_ms}ms"
        return self


class HttpClient:
    """HTTP客户端封装"""
    
    def __init__(self, base_url: str = "", timeout: int = 30,
                 retry_count: int = 3, verify_ssl: bool = True):
        self.base_url = base_url
        self.timeout = timeout
        self.session = requests.Session()
        
        # 重试策略
        retry_strategy = Retry(
            total=retry_count,
            backoff_factor=1,
            status_forcelist=[429, 500, 502, 503, 504],
            allowed_methods=["HEAD", "GET", "POST", "PUT", "DELETE", "OPTIONS", "TRACE"]
        )
        adapter = HTTPAdapter(max_retries=retry_strategy)
        self.session.mount("http://", adapter)
        self.session.mount("https://", adapter)
        
        # SSL验证
        self.session.verify = verify_ssl
    
    def set_auth_token(self, token: str):
        """设置认证Token"""
        self.session.headers.update({"Authorization": f"Bearer {token}"})
    
    def set_default_headers(self, headers: Dict[str, str]):
        """设置默认请求头"""
        self.session.headers.update(headers)
    
    def _build_url(self, path: str) -> str:
        """构建完整URL"""
        if path.startswith("http"):
            return path
        return f"{self.base_url.rstrip('/')}/{path.lstrip('/')}"
    
    def _request(self, method: str, url: str, **kwargs) -> HttpResponse:
        """通用请求方法"""
        full_url = self._build_url(url)
        kwargs.setdefault('timeout', self.timeout)
        
        start = time.time()
        try:
            response = self.session.request(method, full_url, **kwargs)
            elapsed = (time.time() - start) * 1000
            logger.info(f"{method} {full_url} -> {response.status_code} ({elapsed:.0f}ms)")
            
            return HttpResponse(
                status_code=response.status_code,
                body=response.text,
                headers=dict(response.headers),
                response_time_ms=elapsed,
                raw_response=response
            )
        except requests.exceptions.Timeout:
            elapsed = (time.time() - start) * 1000
            raise requests.exceptions.Timeout(
                f"请求超时: {method} {full_url} (耗时{elapsed:.0f}ms, 超时阈值{self.timeout}s)"
            )
    
    def get(self, path: str, params: dict = None, headers: dict = None) -> HttpResponse:
        return self._request("GET", path, params=params, headers=headers)
    
    def post(self, path: str, json_data: dict = None, data: dict = None,
             headers: dict = None) -> HttpResponse:
        return self._request("POST", path, json=json_data, data=data, headers=headers)
    
    def put(self, path: str, json_data: dict = None, headers: dict = None) -> HttpResponse:
        return self._request("PUT", path, json=json_data, headers=headers)
    
    def delete(self, path: str, headers: dict = None) -> HttpResponse:
        return self._request("DELETE", path, headers=headers)
    
    def upload(self, path: str, file_path: str, file_key: str = "file",
               data: dict = None) -> HttpResponse:
        """文件上传"""
        with open(file_path, 'rb') as f:
            files = {file_key: (file_path.split('/')[-1], f)}
            return self._request("POST", path, files=files, data=data)
    
    def close(self):
        self.session.close()
```

## 2. 数据库操作封装

### 通俗释义

数据库操作封装让测试代码能像操作"遥控器"一样操作数据库——不需要手写连接管理、事务控制、资源释放，只需关注SQL和数据。它支持多数据源切换、查询结果自动映射、连接池管理等企业级特性。

### 详细解释

#### Java Spring JdbcTemplate封装

```java
// DatabaseHelper.java
import org.springframework.jdbc.core.JdbcTemplate;
import org.springframework.jdbc.core.RowMapper;
import org.springframework.jdbc.datasource.DriverManagerDataSource;
import javax.sql.DataSource;
import java.util.List;
import java.util.Map;

public class DatabaseHelper {
    
    private final Map<String, JdbcTemplate> templates = new ConcurrentHashMap<>();
    private final Map<String, DataSource> dataSources = new ConcurrentHashMap<>();
    
    /**
     * 注册多数据源
     */
    public void registerDataSource(String name, String url, String username, 
                                   String password, String driverClass) {
        DriverManagerDataSource ds = new DriverManagerDataSource();
        ds.setUrl(url);
        ds.setUsername(username);
        ds.setPassword(password);
        ds.setDriverClassName(driverClass);
        
        // 连接池配置（HikariCP推荐）
        HikariConfig hikariConfig = new HikariConfig();
        hikariConfig.setDataSource(ds);
        hikariConfig.setMaximumPoolSize(20);
        hikariConfig.setMinimumIdle(5);
        hikariConfig.setConnectionTimeout(30000);
        hikariConfig.setIdleTimeout(600000);
        
        HikariDataSource hikariDS = new HikariDataSource(hikariConfig);
        dataSources.put(name, hikariDS);
        templates.put(name, new JdbcTemplate(hikariDS));
    }
    
    public JdbcTemplate getTemplate(String dsName) {
        return templates.get(dsName);
    }
    
    // ===== 查询操作 =====
    
    /** 查询单条记录 */
    public Map<String, Object> queryOne(String dsName, String sql, Object... params) {
        List<Map<String, Object>> results = templates.get(dsName)
            .queryForList(sql, params);
        if (results.isEmpty()) return null;
        if (results.size() > 1) {
            throw new RuntimeException("期望1条记录，实际返回" + results.size() + "条");
        }
        return results.get(0);
    }
    
    /** 查询列表 */
    public List<Map<String, Object>> queryList(String dsName, String sql, Object... params) {
        return templates.get(dsName).queryForList(sql, params);
    }
    
    /** 查询计数 */
    public long queryCount(String dsName, String sql, Object... params) {
        Long count = templates.get(dsName).queryForObject(sql, Long.class, params);
        return count != null ? count : 0L;
    }
    
    /** 查询单值 */
    public <T> T queryForObject(String dsName, String sql, Class<T> type, Object... params) {
        return templates.get(dsName).queryForObject(sql, type, params);
    }
    
    // ===== 更新操作 =====
    
    /** 执行更新（INSERT/UPDATE/DELETE） */
    public int update(String dsName, String sql, Object... params) {
        return templates.get(dsName).update(sql, params);
    }
    
    /** 批量更新 */
    public int[] batchUpdate(String dsName, String sql, List<Object[]> batchParams) {
        return templates.get(dsName).batchUpdate(sql, batchParams);
    }
    
    // ===== 数据校验辅助方法 =====
    
    /** 验证记录存在 */
    public boolean recordExists(String dsName, String tableName, 
                               String whereClause, Object... params) {
        String sql = String.format("SELECT COUNT(1) FROM %s WHERE %s", tableName, whereClause);
        return queryCount(dsName, sql, params) > 0;
    }
    
    /** 验证字段值 */
    public Object getFieldValue(String dsName, String tableName, String fieldName,
                                String whereClause, Object... params) {
        String sql = String.format("SELECT %s FROM %s WHERE %s", 
            fieldName, tableName, whereClause);
        return queryForObject(dsName, sql, Object.class, params);
    }
    
    /** 安全清理测试数据 */
    public int cleanTestData(String dsName, String tableName, String condition) {
        String sql = String.format("DELETE FROM %s WHERE %s", tableName, condition);
        return update(dsName, sql);
    }
}
```

#### Python数据库封装

```python
# db_helper.py
import pymysql
from dbutils.pooled_db import PooledDB
from typing import List, Dict, Any, Optional, Tuple
from contextlib import contextmanager
import logging

logger = logging.getLogger(__name__)

class DatabaseHelper:
    """数据库操作封装（支持多数据源）"""
    
    def __init__(self):
        self._pools: Dict[str, PooledDB] = {}
        self._connections: Dict[str, Any] = {}
    
    def register_datasource(self, name: str, host: str, port: int = 3306,
                           database: str = "", user: str = "", 
                           password: str = "", **kwargs):
        """注册数据源"""
        self._pools[name] = PooledDB(
            creator=pymysql,
            maxconnections=kwargs.get('max_connections', 10),
            mincached=kwargs.get('min_cached', 2),
            maxcached=kwargs.get('max_cached', 5),
            blocking=True,
            host=host,
            port=port,
            user=user,
            password=password,
            database=database,
            charset=kwargs.get('charset', 'utf8mb4'),
            cursorclass=pymysql.cursors.DictCursor
        )
        logger.info(f"数据源注册成功: {name} -> {host}:{port}/{database}")
    
    @contextmanager
    def get_connection(self, ds_name: str):
        """获取数据库连接（上下文管理器，自动释放）"""
        if ds_name not in self._pools:
            raise ValueError(f"数据源未注册: {ds_name}")
        
        conn = self._pools[ds_name].connection()
        try:
            yield conn
            conn.commit()
        except Exception:
            conn.rollback()
            raise
        finally:
            conn.close()
    
    def query_one(self, ds_name: str, sql: str, params: tuple = None) -> Optional[Dict]:
        """查询单条记录"""
        with self.get_connection(ds_name) as conn:
            with conn.cursor() as cursor:
                cursor.execute(sql, params)
                return cursor.fetchone()
    
    def query_all(self, ds_name: str, sql: str, params: tuple = None) -> List[Dict]:
        """查询所有记录"""
        with self.get_connection(ds_name) as conn:
            with conn.cursor() as cursor:
                cursor.execute(sql, params)
                return cursor.fetchall()
    
    def query_count(self, ds_name: str, table: str, 
                    where: str = "1=1", params: tuple = None) -> int:
        """查询记录数"""
        sql = f"SELECT COUNT(1) as cnt FROM {table} WHERE {where}"
        result = self.query_one(ds_name, sql, params)
        return result['cnt'] if result else 0
    
    def execute(self, ds_name: str, sql: str, params: tuple = None) -> int:
        """执行更新语句"""
        with self.get_connection(ds_name) as conn:
            with conn.cursor() as cursor:
                affected = cursor.execute(sql, params)
                return affected
    
    def execute_many(self, ds_name: str, sql: str, params_list: List[tuple]) -> int:
        """批量执行"""
        with self.get_connection(ds_name) as conn:
            with conn.cursor() as cursor:
                affected = cursor.executemany(sql, params_list)
                return affected
    
    def record_exists(self, ds_name: str, table: str,
                      where: str, params: tuple = None) -> bool:
        """校验记录存在性"""
        return self.query_count(ds_name, table, where, params) > 0
    
    def clean_table(self, ds_name: str, table: str, 
                    condition: str = "1=1") -> int:
        """清理表数据（测试专用）"""
        sql = f"DELETE FROM {table} WHERE {condition}"
        return self.execute(ds_name, sql)
    
    def truncate_table(self, ds_name: str, table: str):
        """清空表"""
        self.execute(ds_name, f"TRUNCATE TABLE {table}")
```

```java
public class ExcelHelper {
    
    /**
     * 读取Excel为List<Map>（用于数据驱动测试）
     * 第一行为表头
     */
    public static List<Map<String, String>> readAsMap(String filePath, String sheetName) {
        List<Map<String, String>> result = new ArrayList<>();
        
        try (FileInputStream fis = new FileInputStream(filePath);
             Workbook workbook = new XSSFWorkbook(fis)) {
            
            Sheet sheet = sheetName != null ? 
                workbook.getSheet(sheetName) : workbook.getSheetAt(0);
            
            // 第一行作为表头
            Row headerRow = sheet.getRow(0);
            List<String> headers = new ArrayList<>();
            for (Cell cell : headerRow) {
                headers.add(cell.getStringCellValue());
            }
            
            // 数据行
            for (int i = 1; i <= sheet.getLastRowNum(); i++) {
                Row row = sheet.getRow(i);
                if (row == null) continue;
                
                Map<String, String> rowData = new LinkedHashMap<>();
                for (int j = 0; j < headers.size(); j++) {
                    Cell cell = row.getCell(j, Row.MissingCellPolicy.CREATE_NULL_AS_BLANK);
                    rowData.put(headers.get(j), getCellValueAsString(cell));
                }
                result.add(rowData);
            }
        } catch (IOException e) {
            throw new RuntimeException("读取Excel失败: " + filePath, e);
        }
        return result;
    }
    
    /**
     * 读取Excel为二维数组（TestNG DataProvider格式）
     */
    public static Object[][] readForDataProvider(String filePath) {
        List<Map<String, String>> data = readAsMap(filePath, null);
        Object[][] result = new Object[data.size()][1];
        for (int i = 0; i < data.size(); i++) {
            result[i][0] = data.get(i);
        }
        return result;
    }
    
    /**
     * 写入Excel
     */
    public static void writeData(String filePath, String sheetName,
                                 List<String> headers, List<List<String>> data) {
        try (Workbook workbook = new XSSFWorkbook()) {
            Sheet sheet = workbook.createSheet(sheetName);
            
            // 写表头
            Row headerRow = sheet.createRow(0);
            for (int i = 0; i < headers.size(); i++) {
                headerRow.createCell(i).setCellValue(headers.get(i));
            }
            
            // 写数据
            for (int i = 0; i < data.size(); i++) {
                Row row = sheet.createRow(i + 1);
                List<String> rowData = data.get(i);
                for (int j = 0; j < rowData.size(); j++) {
                    row.createCell(j).setCellValue(rowData.get(j));
                }
            }
            
            try (FileOutputStream fos = new FileOutputStream(filePath)) {
                workbook.write(fos);
            }
        } catch (IOException e) {
            throw new RuntimeException("写入Excel失败: " + filePath, e);
        }
    }
    
    private static String getCellValueAsString(Cell cell) {
        switch (cell.getCellType()) {
            case STRING: return cell.getStringCellValue();
            case NUMERIC: 
                if (DateUtil.isCellDateFormatted(cell)) {
                    return cell.getDateCellValue().toString();
                }
                // 避免科学计数法
                double val = cell.getNumericCellValue();
                if (val == Math.floor(val) && !Double.isInfinite(val)) {
                    return String.valueOf((long) val);
                }
                return String.valueOf(val);
            case BOOLEAN: return String.valueOf(cell.getBooleanCellValue());
            case FORMULA: return cell.getCellFormula();
            default: return "";
        }
    }
```

```java
// CryptoHelper.java
import javax.crypto.Cipher;
import javax.crypto.Mac;
import javax.crypto.spec.IvParameterSpec;
import javax.crypto.spec.SecretKeySpec;
import java.nio.charset.StandardCharsets;
import java.security.*;
import java.security.spec.PKCS8EncodedKeySpec;
import java.security.spec.X509EncodedKeySpec;
import java.util.Base64;

public class CryptoHelper {
    
    // ===== 哈希摘要 =====
    
    /** MD5 */
    public static String md5(String input) {
        try {
            MessageDigest md = MessageDigest.getInstance("MD5");
            byte[] digest = md.digest(input.getBytes(StandardCharsets.UTF_8));
            return bytesToHex(digest);
        } catch (NoSuchAlgorithmException e) {
            throw new RuntimeException(e);
        }
    }
    
    /** SHA-256 */
    public static String sha256(String input) {
        try {
            MessageDigest md = MessageDigest.getInstance("SHA-256");
            byte[] digest = md.digest(input.getBytes(StandardCharsets.UTF_8));
            return bytesToHex(digest);
        } catch (NoSuchAlgorithmException e) {
            throw new RuntimeException(e);
        }
    }
    
    // ===== Base64编解码 =====
    
    public static String base64Encode(String input) {
        return Base64.getEncoder().encodeToString(input.getBytes(StandardCharsets.UTF_8));
    }
    
    public static String base64Decode(String input) {
        return new String(Base64.getDecoder().decode(input), StandardCharsets.UTF_8);
    }
    
    // ===== AES对称加密 =====
    
    private static final String AES_ALGORITHM = "AES/CBC/PKCS5Padding";
    
    /** AES加密 */
    public static String aesEncrypt(String plainText, String key, String iv) {
        try {
            SecretKeySpec keySpec = new SecretKeySpec(
                key.getBytes(StandardCharsets.UTF_8), "AES");
            IvParameterSpec ivSpec = new IvParameterSpec(
                iv.getBytes(StandardCharsets.UTF_8));
            
            Cipher cipher = Cipher.getInstance(AES_ALGORITHM);
            cipher.init(Cipher.ENCRYPT_MODE, keySpec, ivSpec);
            byte[] encrypted = cipher.doFinal(plainText.getBytes(StandardCharsets.UTF_8));
            return Base64.getEncoder().encodeToString(encrypted);
        } catch (Exception e) {
            throw new RuntimeException("AES加密失败", e);
        }
    }
    
    /** AES解密 */
    public static String aesDecrypt(String encryptedText, String key, String iv) {
        try {
            SecretKeySpec keySpec = new SecretKeySpec(
                key.getBytes(StandardCharsets.UTF_8), "AES");
            IvParameterSpec ivSpec = new IvParameterSpec(
                iv.getBytes(StandardCharsets.UTF_8));
            
            Cipher cipher = Cipher.getInstance(AES_ALGORITHM);
            cipher.init(Cipher.DECRYPT_MODE, keySpec, ivSpec);
            byte[] decrypted = cipher.doFinal(Base64.getDecoder().decode(encryptedText));
            return new String(decrypted, StandardCharsets.UTF_8);
        } catch (Exception e) {
            throw new RuntimeException("AES解密失败", e);
        }
    }
    
    // ===== HMAC签名 =====
    
    /** HMAC-SHA256签名（常用于API签名） */
    public static String hmacSha256(String data, String secret) {
        try {
            Mac mac = Mac.getInstance("HmacSHA256");
            SecretKeySpec keySpec = new SecretKeySpec(
                secret.getBytes(StandardCharsets.UTF_8), "HmacSHA256");
            mac.init(keySpec);
            byte[] result = mac.doFinal(data.getBytes(StandardCharsets.UTF_8));
            return bytesToHex(result);
        } catch (Exception e) {
            throw new RuntimeException("HMAC签名失败", e);
        }
    }
    
    /** 构建签名源串（字典序排序参数） */
    public static String buildSignSource(Map<String, String> params) {
        TreeMap<String, String> sorted = new TreeMap<>(params);
        StringBuilder sb = new StringBuilder();
        sorted.forEach((k, v) -> {
            if (v != null && !v.isEmpty()) {
                sb.append(k).append("=").append(v).append("&");
            }
        });
        if (sb.length() > 0) sb.deleteCharAt(sb.length() - 1);
        return sb.toString();
    }
    
    // ===== RSA非对称加密 =====
    
    /** RSA私钥签名 */
    public static String rsaSign(String data, String privateKeyStr) {
        try {
            byte[] keyBytes = Base64.getDecoder().decode(privateKeyStr);
            PKCS8EncodedKeySpec keySpec = new PKCS8EncodedKeySpec(keyBytes);
            KeyFactory keyFactory = KeyFactory.getInstance("RSA");
            PrivateKey privateKey = keyFactory.generatePrivate(keySpec);
            
            Signature signature = Signature.getInstance("SHA256withRSA");
            signature.initSign(privateKey);
            signature.update(data.getBytes(StandardCharsets.UTF_8));
            return Base64.getEncoder().encodeToString(signature.sign());
        } catch (Exception e) {
            throw new RuntimeException("RSA签名失败", e);
        }
    }
    
    /** RSA公钥验签 */
    public static boolean rsaVerify(String data, String signatureStr, String publicKeyStr) {
        try {
            byte[] keyBytes = Base64.getDecoder().decode(publicKeyStr);
            X509EncodedKeySpec keySpec = new X509EncodedKeySpec(keyBytes);
            KeyFactory keyFactory = KeyFactory.getInstance("RSA");
            PublicKey publicKey = keyFactory.generatePublic(keySpec);
            
            Signature signature = Signature.getInstance("SHA256withRSA");
            signature.initVerify(publicKey);
            signature.update(data.getBytes(StandardCharsets.UTF_8));
            return signature.verify(Base64.getDecoder().decode(signatureStr));
        } catch (Exception e) {
            throw new RuntimeException("RSA验签失败", e);
        }
    }
    
    // ===== 辅助方法 =====
    
    private static String bytesToHex(byte[] bytes) {
        StringBuilder sb = new StringBuilder();
        for (byte b : bytes) {
            sb.append(String.format("%02x", b));
        }
        return sb.toString();
    }
}
```

### 案例1：API签名计算工具

**背景：** 被测系统的所有API调用都需要携带MD5或HMAC签名，需要在测试框架中统一计算。

```java
public class ApiSignHelper {
    
    /**
     * 通用签名计算
     * 规则：所有参数按key字典序排序 -> 拼接成key=value&key=value -> 追加secret -> MD5
     */
    public static String calculateSign(Map<String, String> params, String secret) {
        // 1. 排除sign字段本身、空值字段
        TreeMap<String, String> sorted = new TreeMap<>();
        params.forEach((k, v) -> {
            if (!"sign".equals(k) && v != null && !v.isEmpty()) {
                sorted.put(k, v);
            }
        });
        
        // 2. 拼接待签名字符串
        StringBuilder sb = new StringBuilder();
        sorted.forEach((k, v) -> sb.append(k).append("=").append(v).append("&"));
        sb.append("secret=").append(secret);
        
        // 3. MD5签名
        return CryptoHelper.md5(sb.toString()).toUpperCase();
    }
    
    /**
     * 验证响应签名
     */
    public static boolean verifyResponseSign(Map<String, String> response, 
                                            String expectedSign, String secret) {
        String calculated = calculateSign(response, secret);
        return calculated.equals(expectedSign);
    }
    
    /**
     * 时间戳+Nonce防重放
     */
    public static Map<String, String> buildAuthParams(String appId, String secret) {
        Map<String, String> params = new LinkedHashMap<>();
        params.put("app_id", appId);
        params.put("timestamp", String.valueOf(System.currentTimeMillis() / 1000));
        params.put("nonce", UUID.randomUUID().toString().replace("-", ""));
        params.put("sign", calculateSign(params, secret));
        return params;
    }
}
```

### 案例2：数据库数据比对工具

**背景：** 测试中经常需要验证接口操作后数据库数据是否正确更新。

```python
# db_assert.py
from typing import Dict, List, Any

class DatabaseAsserter:
    """数据库断言工具"""
    
    def __init__(self, db_helper):
        self.db = db_helper
        self.ds = "default"
    
    def assert_record_exists(self, table: str, where: str, 
                            params: tuple = None, message: str = ""):
        """断言记录存在"""
        exists = self.db.record_exists(self.ds, table, where, params)
        assert exists, message or f"表中应存在记录: {table} WHERE {where}"
    
    def assert_record_not_exists(self, table: str, where: str,
                                 params: tuple = None, message: str = ""):
        """断言记录不存在"""
        exists = self.db.record_exists(self.ds, table, where, params)
        assert not exists, message or f"表中不应存在记录: {table} WHERE {where}"
    
    def assert_field_equals(self, table: str, field: str, expected: Any,
                           where: str, params: tuple = None):
        """断言字段值等于期望值"""
        actual = self.db.get_field_value(self.ds, table, field, where, params)
        assert actual == expected, \
            f"字段值不匹配: {table}.{field} 期望={expected} 实际={actual}"
    
    def assert_record_count(self, table: str, expected_count: int,
                           where: str = "1=1", params: tuple = None):
        """断言记录数量"""
        actual = self.db.query_count(self.ds, table, where, params)
        assert actual == expected_count, \
            f"记录数不匹配: {table} 期望={expected_count} 实际={actual}"
    
    def assert_record_matches(self, table: str, expected: Dict,
                             where: str, params: tuple = None):
        """断言记录内容匹配"""
        record = self.db.query_one(self.ds, 
            f"SELECT * FROM {table} WHERE {where}", params)
        assert record is not None, f"记录不存在: {table} WHERE {where}"
        
        mismatches = []
        for key, expected_val in expected.items():
            actual_val = record.get(key)
            if actual_val != expected_val:
                mismatches.append(f"  {key}: 期望={expected_val} 实际={actual_val}")
        
        assert not mismatches, \
            f"记录内容不匹配:\n" + "\n".join(mismatches)
```

### 案例5：缓存管理工具

**背景：** 测试中需要清理Redis缓存、验证缓存命中情况。

```java
public class CacheHelper {
    
    private final JedisPool jedisPool;
    
    public CacheHelper(String host, int port, String password) {
        JedisPoolConfig config = new JedisPoolConfig();
        config.setMaxTotal(20);
        config.setMaxIdle(5);
        config.setMinIdle(2);
        config.setTestOnBorrow(true);
        
        this.jedisPool = password != null && !password.isEmpty() ?
            new JedisPool(config, host, port, 3000, password) :
            new JedisPool(config, host, port, 3000);
    }
    
    /** 设置缓存 */
    public void set(String key, String value, int expireSeconds) {
        try (Jedis jedis = jedisPool.getResource()) {
            jedis.setex(key, expireSeconds, value);
        }
    }
    
    /** 获取缓存 */
    public String get(String key) {
        try (Jedis jedis = jedisPool.getResource()) {
            return jedis.get(key);
        }
    }
    
    /** 验证缓存存在 */
    public boolean cacheExists(String key) {
        try (Jedis jedis = jedisPool.getResource()) {
            return jedis.exists(key);
        }
    }
    
    /** 验证缓存过期 */
    public boolean cacheExpired(String key) {
        try (Jedis jedis = jedisPool.getResource()) {
            return jedis.ttl(key) == -2; // -2表示key不存在
        }
    }
    
    /** 批量清理缓存 */
    public void clearByPattern(String pattern) {
        try (Jedis jedis = jedisPool.getResource()) {
            Set<String> keys = jedis.keys(pattern);
            if (!keys.isEmpty()) {
                jedis.del(keys.toArray(new String[0]));
            }
        }
    }
    
    /** 清理所有测试缓存 */
    public void clearTestCache() {
        clearByPattern("test:*");
    }
}
```

## 本章小结

| 封装层 | 核心能力 | 关键设计原则 |
|--------|----------|-------------|
| HTTP客户端 | 统一请求/响应、自动重试、SSL处理、连接池 | Builder模式、拦截器链、统一响应模型 |
| 数据库操作 | 多数据源、查询封装、事务管理、连接池 | 模板方法、资源自动释放、数据校验 |
| 文件操作 | Excel/CSV/JSON/PDF读写、路径管理 | 统一接口、编码处理、数据驱动 |
| 日志工具 | 分级日志、步骤记录、自动归档、脱敏 | 分层输出、结构化日志、性能优化 |
| 加解密工具 | MD5/SHA/AES/RSA、签名验签 | 算法抽象、密钥外部化、性能缓存 |
| 消息通知 | 邮件/钉钉/企微/Slack、模板化 | 统一接口、多渠道分发、异步发送 |

> 工具封装层的核心哲学是：**让测试代码只描述"测什么"和"怎么断"，而不是"怎么建连接"、"怎么写签名"、"怎么读文件"。** 好的工具封装让测试用例代码像自然语言一样可读。
