# 12_环境搭建零基础避坑手册

## 模块简介

本模块为零基础测试工程师提供测试环境搭建的系统化指南，涵盖标准化搭建流程、依赖配置管理和常见环境问题排查，手把手教你从零构建稳定、可复现的测试环境，大幅减少因环境问题导致的无效工作时间。

---

## 目录结构

```
12_环境搭建零基础避坑手册/
├── README.md              # 本文件
├── 01_环境搭建规范.md      # 操作系统、语言运行时、数据库、中间件的标准化安装配置
├── 02_依赖配置.md          # 测试框架、工具库、驱动的版本管理与锁定
└── 03_环境问题汇总.md      # 环境相关常见报错、诊断方法和解决方案
```

---

## 学习路线图

```
第1步: 01_环境搭建规范      按规范完成基础软件栈的安装和配置
  │
第2步: 02_依赖配置          学会管理测试项目的依赖版本和来源
  │
第3步: 03_环境问题汇总      遇到问题时快速查阅，掌握排查思路
```

---

## 学习建议

1. 按顺序学习：先搭建环境基础，再管理依赖，最后掌握排查能力
2. 动手实践：边学边搭建，将命令和配置亲自执行一遍
3. 将 01（环境搭建规范）作为日常搭建的检查清单
4. 参考本模块 03_环境问题汇总.md 快速解决环境相关报错
5. 建议学习周期：3-5天
   - Day 1-2：01（环境搭建）
   - Day 3：02（依赖配置）
   - Day 4-5：03（问题汇总+实践巩固）
